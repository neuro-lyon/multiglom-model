Scripts
=======

Run a single simulation
-----------------------

In order to run a single simulation you may use the script
``multiglom_network.py``. Usage: ``python multiglom_network.py <parameter_file>``
, see more options by using ``multiglom_network.py --help``.

Below is an summary of what the script does and it what order.
More information is given throughout this script, check the source ;) .

.. automodule:: multiglom_network


Run multiple simulations in parallel
------------------------------------

In order to explore a parameter space it is easier to run a lot of simulation
and then look at the outputs. This is what the script ``run_multip.py`` is for.
